# Ncu(npm-check-updates)

[npm-check-updates](https://github.com/tjunnone/npm-check-updates)

package.json  
bower.json  

最新版をチェックするツール


## インストール

    sudo npm install -g npm-check-updates
    


## チェック


    ncu

    
## チェック+アップグレード

    ncu -u
    

package.json が書き換わります


＊ gulp module がチェックできない様子？


**global check**

    ncu -g
