ブラウザ

  • Internet Explorer 11.x 以降
  • Firefox 最新版
  • Google Chrome 最新版
  • Safari 8.x 以降
  • Microsoft Edge 最新版

OS

  • iOS 8.4.1 以降
  • Android 4.2 以降
